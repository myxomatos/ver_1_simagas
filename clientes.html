Clientes

GET / POST / PUT para tabla vencli.

Lista de clientes

GET /api/vencli

    Buscar cliente

    GET /api/vencli/:cli_llave

    Escribe una llave y presiona “Buscar”.
    Tip: Al buscar, se autollenará el formulario de “Actualizar”.

    Crear cliente

    POST /api/vencli

    —

    Actualizar cliente

    PUT /api/vencli/:cli_llave (rechaza cli_saldo)

    Esta llave NO se cambia, solo identifica qué cliente actualizar.
    Puedes mandar cualquier campo editable. Si mandas cli_saldo el server rechazará.
